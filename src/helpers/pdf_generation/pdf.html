Fase de incio

Nombre: Proceso de la Fase incial

Objetivo Conocer las necesidades del cliente

Inidicadores y métricas: Estimación de tiempo

Descripción de actividades
• Realizar un borrador sobre el tipo de preguntas que se le harán conocer al cliente.
• Una vez terminado, hablar con el Scrum Master para conocer si está de acuerdo con el cuestionario.
• Si en dado caso una pregunta es incorrecta o fuera de lo que queremos conocer, se realizarán cambios.
• Por último, se hablará con el cliente, con el propósito de realizar el cuestionario de forma espontánea y así conocer como estará enfocado el proyecto y los requerimientos que este tendrá.
